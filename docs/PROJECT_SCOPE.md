# BudgetBuddy

## Project Overview

BudgetBuddy is a Full-Stack Personal Budget Planning and Expense Management Platform developed using Django REST Framework and React.

The application helps users manage their personal finances by tracking income, expenses, budgets, and savings goals. It also provides reports and notifications to help users make better financial decisions.

---

## Objectives

- Manage personal income
- Track daily expenses
- Create monthly budgets
- Monitor savings goals
- Generate financial reports
- Receive spending notifications
